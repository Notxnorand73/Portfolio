project description

scorify is a command line tool written in python designed for managing and exporting sales data.


download scorify.py
usage instructions
  1. ensure you have python installed on your system.
  2. download the scorify.py file using the link above.
  3. open your command line interface and navigate to the directory where scorify.py is located.
  4. run the script using the command: python scorify.py
  5. follow the on-screen prompts to input and manage your sales data.
documentation

Commands:

  • sell price: add a new sales record.
  • sold record/id: mark a sales record as sold.
  • refund record/id: remove a sales record.
  • list: display all sales records.
  • update: reload data from the JSON file.
  • export format: export data in the specified format (csv, txt, json, sd -scorify document).
  • exit: save changes and exit the application.
